## Install

```
wget https://github.com/k-shino/mago-talk/raw/master/setup.sh
chmod +x ./setup.sh
./setup.sh

sudo ./raspaas/bootstrap.sh mago-talk
cd ./mago-talk/
mago-talk create
git push mago-talk master
```
